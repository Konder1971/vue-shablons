Очистить кэш npm:

Установить менеджер версий NodeJS: 
npm install -g n

Установить версию ноды:

Стабильная версия:
n stable

Последняя версия:
n latest

Установка определённой версии:
n 0.8.14

Удаление определённой версии:
n rm 0.8.14
Или:
n - 0.8.14
